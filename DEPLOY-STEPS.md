# GitHub Repository Setup

## วิธีสร้าง Repository

1. เปิดลิงก์นี้: https://github.com/new

2. กรอกข้อมูล:
   - Repository name: `myhome-diy`
   - Description: `Myhome DIY - แต่งบ้าน ปลูกทุกสวย เลี้ยงลูก ใช้ของดี`
   - เลือก: **Public** (ฟรีและใช้ GitHub Pages ได้)
   - ❌ ไม่ต้องเลือก "Add README" (เรามีแล้ว)
   - ❌ ไม่ต้องเลือก .gitignore (เรามีแล้ว)
   - ❌ ไม่ต้องเลือก license

3. คลิก **Create repository**

4. เสร็จแล้ว! คัดลอก URL: `https://github.com/chkm/myhome-diy.git`

---

## หลังจากสร้าง Repository แล้ว

กลับมาที่ Terminal แล้วพิมพ์ว่า "สร้างแล้ว" เพื่อทำขั้นตอนถัดไป
