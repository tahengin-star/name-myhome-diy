# 🚀 Quick Start Guide - เริ่มใช้งาน Chat กับ Hermes Agent

## ขั้นตอนง่ายๆ 3 ขั้นตอน:

### 1️⃣ ติดตั้ง Dependencies (ครั้งแรกเท่านั้น)
```powershell
pip install flask flask-cors
```

### 2️⃣ เริ่ม Chat Server
```powershell
cd C:\Users\tah_e\Hermes-Agent\Myhome-DIY-Website
python hermes_chat_server.py
```

คุณจะเห็น:
```
============================================================
🤖 Hermes Agent Chat API Server
============================================================
✅ Server is ready!
============================================================
```

### 3️⃣ เปิดหน้า Chat
- ดับเบิลคลิกที่ไฟล์ `chat.html`
- หรือเปิดเบราว์เซอร์ไปที่ `index.html` แล้วคลิก "💬 AI Chat"
